step 1 fork the source code on github
step 2 open it in your IDE of choice with the discord.py and requests modules installed (if not, pip install discord requests)
step 3 get your discord api token on the discord developer portal and invite the bot to your server
step 4 paste the api token into the token.txt file
step 5 run the python file and test it (type !catimg into your discord server)
